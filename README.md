# 🛒 Blinkit Data Analysis Dashboard

## 📊 Tools Used

* Power BI
* Excel

## 📌 Project Overview

This project analyzes Blinkit grocery sales data to understand outlet performance, product trends, and customer preferences.

## 🔍 Key Insights

* Tier 1 cities have highest sales
* Fruits and Snacks category perform best
* Medium size outlets generate more revenue
* High fat products contribute more to total sales

## 📈 Dashboard Features

* Sales by Outlet Size
* Sales by Location Type
* Product Category Analysis
* Outlet Establishment Trends



## 📁 Dataset

Contains sales data of Blinkit outlets including:

* Item Type
* Fat Content
* Outlet Size
* Sales Amount
## 🚀 Conclusion
This dashboard helps in identifying high-performing outlets and popular product categories, which can support business decision-making.
